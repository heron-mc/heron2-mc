Heron.controls.ZoomPreviousNext Class

Extends Heron.base.control
Defined in: src\controls\ZoomPreviousNext.js:4
Module: Heron

This control saves the zoom history and has methods to go to previous and next zoom extent. These methods can be called from a Previous and Next button.
type: "zoomprevious"
type: "zoomnext"
